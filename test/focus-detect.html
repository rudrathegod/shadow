Java Question: Reverse Words in a Sentence

Write a Java method with the following signature:

public static String reverseWords(String sentence)

It should reverse the order of words in sentence (words separated by single spaces), without using any built-in reverse utility (e.g. no Collections.reverse). Preserve the original words themselves — only reorder them.

Example: "the quick brown fox" → "fox brown quick the"
